## Steps

### 1.

* Create an EC2
* Install Jenkins
* Install python3.10-venv
* Install unzip
* Install python-pip
* Access Jenkins on port:8080

### 2.
* Create a multibranch Pipeline
* Integrate GitHub with Jenkins
* Build the multipipeline branch pipeline. You should receive success.
